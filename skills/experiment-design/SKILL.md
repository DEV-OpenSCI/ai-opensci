# Experiment Design — 实验方案生成

你将根据研究假设生成一个完整的实验设计方案。

## 用户输入

研究假设: `$ARGUMENTS`

## 执行流程

### Step 1: 解析假设

从用户输入中提取:
- **自变量 (IV)**: 被操控的变量
- **因变量 (DV)**: 被测量的结果
- **控制变量**: 需要保持不变的因素
- **假设方向**: 单尾 vs 双尾

### Step 2: 生成实验设计

根据假设类型选择合适的实验设计:

| 假设类型 | 推荐设计 | 统计方法 |
|---------|---------|---------|
| A vs B 差异 | 随机对照实验 (RCT) | t-test / Mann-Whitney |
| 多组比较 | 析因设计 | ANOVA |
| 因果关系 | 前后对照 + 对照组 | Mixed ANOVA |
| 相关性 | 观察性研究 | 相关/回归 |
| 剂量效应 | 梯度设计 | 回归分析 |

### Step 3: 样本量估算

使用 G*Power 公式的简化版本:
- 设定显著性水平 α = 0.05
- 统计功效 1-β = 0.80
- 预期效应量（小/中/大）
- 计算最小样本量

### Step 4: 输出方案

```markdown
# 实验设计方案

## 研究假设
- H₀ (零假设): ...
- H₁ (备择假设): ...

## 实验设计
- 设计类型: [e.g., 2×3 混合设计]
- 自变量: ...
- 因变量: ...
- 控制变量: ...

## 变量矩阵
| 条件 | IV1-Level1 | IV1-Level2 |
|------|-----------|-----------|
| IV2-Level1 | Cell(1,1) | Cell(1,2) |
| IV2-Level2 | Cell(2,1) | Cell(2,2) |

## 样本量
- 效应量: d = ...
- 最小样本量: N = ... (每组 n = ...)
- 考虑脱落率 (20%): 建议招募 N = ...

## 统计分析计划
1. 描述性统计
2. 正态性检验 (Shapiro-Wilk)
3. 主分析: [具体检验方法]
4. 效应量报告
5. 多重比较校正 (如适用)

## 实验流程 (Protocol)
1. 被试招募
2. 知情同意
3. 随机分组
4. 实验操作
5. 数据采集
6. 数据分析

## 潜在威胁与对策
- [威胁]: [对策]
```
